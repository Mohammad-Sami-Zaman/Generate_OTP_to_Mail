# 📧 Email OTP Verification System

A full-stack **Email OTP Verification System** built using **HTML, CSS, JavaScript, Node.js, Express.js, and Gmail SMTP**.

---

## 🚀 Features

- Send OTP to user email
- OTP verification system
- Expiry-based OTP validation
- No database required (in-memory OTP)
- Secure Gmail SMTP integration
- REST API architecture

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Nodemailer
- Gmail SMTP

---

## ⚙️ How It Works

1. User enters email
2. Server generates OTP
3. OTP sent via Gmail
4. User verifies OTP
5. Verification successful
